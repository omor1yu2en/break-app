# 休憩回し計算アプリ

白い1画面のアプリ。下線の箇所をタップして「開始時刻・総人数・一度に回す人数・休憩分数」を変更すると、合計時間と終了予定時刻が自動で更新されます。

## 起動

Docker（ローカル環境構築済み）で、このフォルダに入って:

```
docker compose up -d
```

ブラウザで http://localhost:8080 を開く。

停止:

```
docker compose down
```

## 構成

- `index.html` — 画面とロジック（Vue 3 を CDN 読み込み。ビルド工程なし）
- `Dockerfile` — nginx で配信
- `docker-compose.yml` — `localhost:8080` 公開

Node も Postgres も使いません。HTMLを直接ブラウザで開いても動きます。

## 操作

- 「今から」をタップ → 時刻入力。空のままなら現在時刻、入力すればその時刻から計算。
- 「5」（総人数）「1」（一度に回す人数）「45」（休憩分数）をそれぞれタップして数値入力。
- Enter または 入力欄の外をクリックで確定。

## 計算ロジック

- 必要回数 ＝ 切り上げ(総人数 ÷ 一度に回す人数)
- 合計分 ＝ 休憩分数 × 必要回数
- 終了時刻 ＝ 開始時刻 ＋ 合計分
- 合計時間の表示: 60分未満は「◯分」、ちょうどは「◯時間」、それ以外は「◯時間◯分」
- 終了時刻が24時を超える場合は mod 24 で繰り越し、「（翌日）」を付与
